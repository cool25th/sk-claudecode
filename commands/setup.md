# SKC Setup

This is the preferred setup command.

Use `/sk-claudecode:setup` to run the setup flow.

## Quick Install

```bash
/sk-claudecode:setup
/sk-claudecode:setup --minimal
/sk-claudecode:setup --full
```

It is functionally equivalent to the previous `setup` workflow and is the canonical command name going forward.
