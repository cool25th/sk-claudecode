# Execution Modes

Reference directory for SKC execution modes. These are not standalone skills — each mode is invoked via its corresponding slash command.

## Available Modes

| Mode | Command | Description |
|------|---------|-------------|
| `autopilot` | `/sk-claudecode:autopilot` | Self-referential loop with architect verification |
| `ecomode` | `/sk-claudecode:ecomode` | Resource-efficient mode |
| `pipeline` | `/sk-claudecode:pipeline` | Sequential multi-step pipeline |
| `ralph` | `/sk-claudecode:ralph` | Iterative loop until task completion |
| `swarm` | `/sk-claudecode:swarm` | N coordinated agents with SQLite-based claiming |
| `ultrapilot` | `/sk-claudecode:ultrapilot` | Parallel autopilot with file ownership partitioning |
| `ultrawork` | `/sk-claudecode:ultrawork` | Maximum performance with parallel agent orchestration |
| `ultraqa` | `/sk-claudecode:ultraqa` | QA cycling: test → verify → fix → repeat |
