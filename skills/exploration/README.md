# Exploration

Reference directory for codebase exploration sub-modes. These are invoked via the `explore` agent or research skills.

## Available Sub-Modes

| Sub-Mode | Purpose |
|----------|---------|
| `analyze` | Deep codebase analysis |
| `deepinit` | Deep initialization and discovery |
| `research` | Research-oriented exploration |
| `ultra-research` | Parallel research with cross-validation |

- `/skill research` ‚Äî Orchestrated parallel research
