# Quality Skills

Code quality, testing, and verification workflow skills.

## Available Modes

| Mode | Purpose |
|------|---------|
| `tdd` | Test-Driven Development workflow |
| `ultratdd` | High-intensity TDD with parallelization |
| `verification` | Build/test/lint verification loop |
| `security-review` | Security-focused code analysis |

## Usage

```bash
/sk-claudecode:quality/tdd
/sk-claudecode:quality/ultratdd
```

- `qa-tester`, `qa-tester-high`
- `tdd-guide`, `tdd-guide-low`
